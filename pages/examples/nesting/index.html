Nested sections

Sections can be nested to arbitrary levels, and Anglebars will figure out what you're referring to. It is thus possible, though not recommended, to recreate the plot of Inception using mustaches. Note that we can still find the value of kick when the current context is the final dream:

var anglebars = new Anglebars({ el: 'output', template: $('#template').html(), data: { reality: { kick: 'However deeply nested we are, we can still get back to the top', level: 'one', setting: 'The Airplane', dream: { level: 'two', setting: 'The City', dreamer: 'Yusuf', dream: { level: 'three', setting: 'The Hotel', dreamer: 'Arthur', dream: { level: 'four', setting: 'The Snow Fortress', dreamer: 'Eames', dream: { level: 'five', setting: 'Limbo', dreamer: 'Cobb' } } } } } } });
