> ### ractive.add( keypath[, number ])
> Returns a `Promise` (see {{{createLink 'Promises'}}})

> > #### **keypath** *`String`*
> > The {{{createLink 'keypaths' 'keypath'}}} of the number we're incrementing, e.g. `count`

> > #### number
> > Defaults to `1`. The number to increment by
